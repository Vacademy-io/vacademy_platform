# Job Descriptions for Vacademy Platform

## 1. Java Full Stack Engineer

**Location:** Remote / [City, Country]  
**Type:** Full-time  
**Experience:** 2-5 Years

### About Vacademy

Vacademy is an AI-enabled, open-source Learning Management System (LMS) built with a modern microservices architecture. We provide comprehensive tools for educational institutions, instructors, and learners, utilizing cutting-edge technologies like AI-powered assessment grading, intelligent chatbots, and multi-channel notifications. We are looking for a passionate Full Stack Engineer to help us build the next generation of ed-tech solutions.

### Role Overview

As a Java Full Stack Engineer, you will be responsible for designing, developing, and maintaining scalable microservices and responsive frontend applications. You will work across the entire technology stack, from database design to UI/UX implementation, ensuring high performance and reliability.

### Key Responsibilities

- **Backend Development:** Design and build robust RESTful APIs using **Java 17** and **Spring Boot** within a distributed microservices ecosystem.
- **Frontend Development:** Develop dynamic and responsive web applications using **React 18+**, **TypeScript**, **Vite**, and **Tailwind CSS**.
- **System Architecture:** Contribute to the architectural decisions, database modeling (**PostgreSQL**), and service-to-service communication strategies.
- **DevOps & Cloud:** Work with **Docker**, **Kubernetes**, and **AWS** services (S3, SES, RDS) to ensure smooth deployment and operation.
- **Code Quality:** Write clean, maintainable code with unit and integration tests. Utilize **GitHub Actions** for CI/CD pipelines.
- **Collaboration:** Work closely with product managers and other engineers to deliver features like AI-powered tools, assessment systems, and real-time notifications.

### Required Skills & Qulifications

- **Java Ecosystem:** Strong proficiency in Java 17+, Spring Boot, Spring Security, and Spring Data JPA.
- **Frontend Technologies:** Solid experience with React, TypeScript, and modern state management (e.g., TanStack Query).
- **Database:** Experience with relational databases (PostgreSQL) and caching mechanisms (Redis).
- **Microservices:** Understanding of microservices architecture, REST APIs, and inter-service communication.
- **Tools:** Familiarity with Git, Maven, Docker, and Linux environments.
- **Problem Solving:** Strong analytical skills and the ability to debug complex issues in a distributed system.

### Nice to Have

- Experience with **Kubernetes** and **Helm**.
- Knowledge of **AWS** services.
- Experience with **Apache Kafka** or other message brokers.
- Familiarity with **AI/ML integration** (OpenAI APIs, Python).

## 2. Java Full Stack Intern

**Location:** Remote / [City, Country]  
**Type:** Internship (3-6 Months)

### About the Role

We are seeking a motivated and eager-to-learn Java Full Stack Intern to join our engineering team. This is a unique opportunity to work on a real-world, open-source EdTech platform, gain hands-on experience with modern technologies, and contribute to meaningful projects.

### What You Will Learn & Do

- Assist in developing new features for both the backend (Spring Boot) and frontend (React/TypeScript).
- Write and maintain code for our microservices (Auth, Assessment, Community, etc.).
- Fix bugs and improve application performance.
- Learn about industry-best practices in CI/CD, containerization (Docker), and cloud deployment.
- Collaborate with senior engineers to understand software design patterns and architecture.

### Requirements

- **Education:** Currently pursuing or recently completed a degree in Computer Science, Engineering, or a related field.
- **Core Java:** Good understanding of Object-Oriented Programming (OOP) and Java fundamentals.
- **Web Basics:** familiarity with HTML, CSS, and JavaScript.
- **Frameworks:** Basic knowledge of **Spring Boot** or **React** is a huge plus.
- **Database:** Basic understanding of SQL and relational databases.
- **Attitude:** A proactive learner with good communication skills and a passion for technology.

### Why Join Us?

- Hands-on mentorship from experienced engineers.
- Exposure to a modern tech stack: Java 17, Spring Boot, React, Docker, Kubernetes.
- Opportunity to contribute to an open-source project impacting education globally.
- Potential for full-time offer based on performance.
